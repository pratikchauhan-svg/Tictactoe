# Tictactoe
This is a simple Tic-Tac-Toe game developed in Python where a player competes against the computer on a 3×3 grid. The player uses "X" while the computer uses "O" and makes random moves. The game automatically checks for winning combinations across rows, columns, and diagonals, and also detects tie situations.
# Tic-Tac-Toe Game in Python

A simple command-line Tic-Tac-Toe game built using Python.

## Description

This project allows a player to play Tic-Tac-Toe against the computer. The player uses "X" and the computer uses "O". The game checks for wins across rows, columns, and diagonals and declares the winner or a tie when the game ends.

## Features

* Player vs Computer gameplay
* Random computer moves
* Win detection
* Tie detection
* Easy-to-understand Python code

## Technologies Used

* Python 3
* Random Module

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/tic-tac-toe-python.git
   ```

2. Open the project folder.

3. Run the program:

   ```bash
   python tic_tac_toe.py
   ```
